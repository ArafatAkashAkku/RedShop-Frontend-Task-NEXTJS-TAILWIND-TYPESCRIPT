## Getting Started

First, clone the project:

```bash
#first
git clone mocktwilight
# or
cd .\mocktwilight\
# or
code -r .\mocktwilight\
```

Second, run the development server:

```bash
#first
npm install
# then
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
